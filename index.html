Privacy Policy

Last updated: 2026

Overview

Silver Chat Studio is a Twitch chat overlay and TTS tool for OBS. The application processes chat messages and optional OAuth data to provide real-time chat display, AI meme processing, and text-to-speech features.

Data We Collect

  • Twitch username and chat messages
  • OAuth authentication tokens (if user connects accounts)
  • Application settings (selected voice, UI preferences)

We do not collect or store sensitive personal data beyond what is required for the application functionality.

How We Use Data

  • Display Twitch chat in OBS overlay
  • Generate TTS audio from chat messages
  • Optional meme AI message transformation
  • Save user preferences locally

Google API Services

Our application may use Google API Services for authentication or related features.

Any use of Google user data strictly follows the Google API Services User Data Policy, including the Limited Use requirements.

We only use Google user data to provide user-facing functionality explicitly requested by the user. We do not:

  • Sell or transfer Google user data to third parties
  • Use Google user data for advertising
  • Use Google user data for profiling or analytics outside app functionality
  • Allow human access to data except for security, debugging with consent, or legal requirements

Data Storage

Most data is stored locally on the user’s device or in local configuration files. We do not maintain a remote database of user content.

Data Retention

Data is kept only as long as necessary for application functionality. Users may delete local data by removing the application configuration.

User Rights

Users can request deletion or modification of stored data by contacting support.
