# Политика конфиденциальности Silver Chat Studio

**Последнее обновление:** 20 мая 2026 г.

## 1. Общие положения

Silver Chat Studio ("приложение") уважает вашу конфиденциальность. Эта политика описывает, какие данные мы собираем и как их используем.

## 2. Какие данные мы собираем

Приложение собирает и хранит **только локально** на вашем компьютере:

- **Twitch OAuth токены** (access token, refresh token) — для подключения к чату Twitch
- **Настройки приложения** (порт WebSocket, настройки звука, триггеры)
- **Имя пользователя Twitch** — для отображения в интерфейсе

## 3. Как мы используем данные

Собранные данные используются **исключительно** для:
- Подключения к Twitch API и чтения чата
- Отображения сообщений в OBS через WebSocket
- Сохранения ваших настроек между запусками программы
- Автообновления токенов доступа

## 4. Хранение данных

- **Все данные хранятся локально** на вашем компьютере в папке программы
- Мы **не передаём** ваши данные третьим лицам
- Мы **не отправляем** данные на наши серверы
- Токены Twitch хранятся в файле `tokens.json` в зашифрованном виде

## 5. Интеграция с Twitch

Приложение использует **Twitch API** для:
- Чтения сообщений чата
- Получения информации о стриме (зрители, категория)
- Отображения аватаров пользователей

Используя приложение, вы соглашаетесь с [Условиями использования Twitch](https://www.twitch.tv/p/legal/terms-of-service/)

## 6. Google OAuth (если используется)

Если вы используете интеграцию с YouTube:
- Приложение запрашивает доступ к вашему Google аккаунту через OAuth 2.0
- Мы получаем только базовую информацию (имя, email, ID канала)
- Данные используются только для авторизации в YouTube API

## 7. Ваши права

Вы имеете право:
- **Удалить все данные** — удалите файл `tokens.json` и `settings.json`
- **Отозвать доступ** — в настройках Twitch: https://www.twitch.tv/settings/connections
- **Деинсталлировать приложение** — просто удалите папку с программой

## 8. Безопасность

Мы принимаем меры для защиты ваших данных:
- Токены хранятся локально, не в облаке
- Соединение с Twitch API использует HTTPS
- Приложение не содержит рекламы и трекеров
