# Политика конфиденциальности — Silver Chat Studio

**Дата вступления в силу:** 28 мая 2026 г.

## 1. Общие положения

Приложение **Silver Chat Studio** («Приложение») уважает вашу конфиденциальность. Этот документ описывает, какие данные мы собираем, как их используем и защищаем.

## 2. Какие данные мы собираем

Приложение собирает и хранит **исключительно локально** на устройстве пользователя:

- **OAuth-токены Twitch** (access token, refresh token) — для подключения к чату и получения информации о стриме
- **Настройки приложения** (порт WebSocket, параметры звука, триггеры реакций, предпочтения интерфейса)
- **Имя пользователя и ID канала Twitch** — для отображения в интерфейсе и формирования запросов к API

## 3. Как мы используем данные

Собранные данные используются **только для функциональности приложения**:

- Подключение к Twitch API для чтения чата и получения информации о стриме
- Передача сообщений в OBS через локальный WebSocket-сервер
- Сохранение пользовательских настроек между сеансами
- Автоматическое обновление токенов доступа при истечении срока действия

## 4. Хранение и защита данных

- **Все данные хранятся локально** в папке приложения на вашем компьютере
- Мы **не передаём** ваши данные третьим лицам, рекламным сетям или аналитическим сервисам
- Мы **не отправляем** данные на внешние сервера, кроме официальных API платформ (Twitch, YouTube и др.)
- Файлы `tokens.json` и `settings.json` содержат чувствительную информацию — не передавайте их другим лицам

## 5. Интеграция с внешними платформами

Приложение использует официальные API:

- **Twitch API** — для чтения чата, получения информации о стриме и аватарах пользователей
- **Google OAuth** (опционально) — для авторизации в YouTube (если функция включена)

Используя Приложение, вы соглашаетесь с условиями использования соответствующих платформ:
- [Twitch Terms of Service](https://www.twitch.tv/p/legal/terms-of-service/)
- [Google Privacy Policy](https://policies.google.com/privacy)

## 6. Ваши права

Вы имеете право:

- **Удалить все данные** — удалите файлы `tokens.json`, `settings.json`, `triggers.json` из папки приложения
- **Отозвать доступ** — в настройках аккаунта платформы: [Twitch Connections](https://www.twitch.tv/settings/connections)
- **Деинсталлировать Приложение** — просто удалите папку с программой

## 7. Безопасность

Мы принимаем разумные меры для защиты ваших данных:

- Токены хранятся локально, без передачи в облако
- Все соединения с внешними API используют HTTPS
- Приложение не содержит рекламы, трекеров или скрытых модулей сбора данных
